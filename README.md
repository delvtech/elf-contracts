### Prerequisites

- [npm](https://nodejs.org/en/download/)

### Setup

```
git clone git@github.com:element-fi/elf-contracts.git
```

```
cd elf-contracts
npm install
```

### Build

```
npm build
```

### Test

```
npm test
```
